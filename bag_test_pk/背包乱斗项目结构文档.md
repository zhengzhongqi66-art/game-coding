## 数据流图

### 游戏启动流程

```
main.gd._ready()
    │
    ├─► _init_item_factory()      → test_items.gd 创建所有物品模板
    ├─► _init_character_templates() → 创建角色模板（修士/守卫/行者）
    ├─► _init_monster_templates()   → 创建怪物模板（史莱姆/狼妖/石像）
    ├─► _setup_nodes()              → 创建背包视图和战斗管理器
    └─► _add_test_items_to_inventories() → 添加初始物品
```

### 战斗流程

```
点击"开始战斗"
    │
    ▼
battle_manager.start_battle(player_data, player_items, monster_data, monster_items)
    │
    ├─► 设置玩家/怪物显示
    ├─► refresh_player_side() → 计算HP、防御、构建 ItemRuntime
    ├─► refresh_enemy_side()  → 同上
    └─► 进入 FIGHTING 状态
          │
          ▼
    _process(delta) 循环
          │
          ├─► 玩家物品冷却累加 → 冷却完毕 → _perform_item_attack()
          │                                    │
          │                                    ▼
          │                         battle_resolver.calculate_damage()
          │                                    │
          │                                    ▼
          │                         defender.take_damage()
          │                                    │
          │                                    ▼
          │                         HP <= 0 ? → 战斗结束
          │
          └─► 怪物物品冷却累加 → 同上
```

## 核心数据结构

### ItemData（物品静态数据）

```gdscript
ItemData:
  id: String                    # 唯一标识
  item_name: String             # 显示名称
  description: String           # 描述
  icon: Texture2D               # 图标
  
  base_grid_size: Vector2i      # 占据格子尺寸 (3, 2)
  rotation: int                 # 旋转角度 0/90/180/270
  max_stack: int                # 最大堆叠数量
  
  attack_bonus: int             # 攻击加成
  defense_bonus: int            # 防御加成
  hp_bonus: int                 # HP加成
  speed_bonus: float            # 攻速加成
  
  behavior_type: String         # weapon_attack / passive_armor / consumable
  base_attack_speed: float      # 基础攻击间隔（秒）
  affects_by_bag_state: bool    # 是否受武器方向影响
  
  use_effect_type: String       # heal / buff_attack
  use_effect_value: int         # 效果数值
```

### ItemRuntime（物品运行时状态）

```gdscript
ItemRuntime:
  data: ItemData                # 引用静态数据
  owner: CharacterData/MonsterData  # 所属角色
  
  current_attack: int           # 当前攻击力（已计算加成）
  current_attack_speed: float   # 当前攻速
  current_behavior_type: String # 行为类型
  cooldown_timer: float         # 冷却计时器
```

### 背包物品实例

```gdscript
{
  "data": ItemData,             # 物品数据
  "count": int,                 # 堆叠数量
  "grid_pos": Vector2i          # 在背包中的位置
}
```

## 计算公式

### 伤害计算

**位置**: `logic/battle_resolver.gd:4`

```gdscript
func calculate_damage(attacker_attack: int, defender_defense: int) -> int:
    return max(1, attacker_attack + randi() % 5 - defender_defense)
```

**示例**:
- 攻击力 30，防御 15 → 伤害 = max(1, 30 + 0~4 - 15) = 15~19

### 攻击力计算

**位置**: `logic/item_state_calculator.gd:31`

```gdscript
current_attack = item.attack_bonus + int(owner.strength * 0.3)
```

### 攻速计算

**位置**: `logic/item_state_calculator.gd:32`

```gdscript
current_attack_speed = max(0.2, item.base_attack_speed - item.speed_bonus - owner.agility * 0.02)
```

### 方向加成

**位置**: `logic/direction_manager.gd:19`

```gdscript
static func get_direction_bonus(weapon_rotation: int) -> float:
    return 1.0 if is_weapon_effective(weapon_rotation) else 0.5
```

| 武器朝向 | 敌方方向 | 伤害倍率 |
|----------|----------|----------|
| 0° (→) | right | 100% |
| 90° (↓) | down | 100% |
| 180° (←) | left | 100% |
| 270° (↑) | up | 100% |
| 其他组合 | - | 50% |

### 总属性计算

player_unit.max_hp = player_base_hp + _sum_hp(player_items)

## 游戏状态

```gdscript
enum GameState { PREPARE, BATTLE, RESULT }
```

| 状态 | 说明 |
|------|------|
| PREPARE | 准备阶段，可整理背包 |
| BATTLE | 战斗进行中 |
